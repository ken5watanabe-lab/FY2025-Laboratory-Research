## 研究テーマ
### 背景・目的
広範なDNAの文脈長と高解像度の解析が可能な遺伝子制御を予測するDNA配列モデルAlphaGenomeが公開された
SNP-遺伝子発現間の解析にAlphaGenomeが有用であるかを検証することが本研究の目的である

### 方法
J-MICC研究のSNPアレイのターゲット塩基情報をもとに，X染色体における
ダミーSNPデータセットを用意し，AlphaGenomeを用いて遺伝子発現量のシミュレーション解析を行った

### 結果・考察
AlphaGenome解析に適するシミュレーションの回数
・30 回ほどで発現予測の精度は高い(RMSE ≒0.01 )
・25 回ほどで個人のSNPが再現されるリスクは低い (p < 0.01 )
SNPデータから遺伝子発現の大規模予測においてAlphaGenomeの有用性が支持された
