# 2025年度 医学研究実習

[🇺🇸 English](README.md) | 🇯🇵 日本語  
本リポジトリには，徳島大学医学部の2025年度医学研究実習において，メディカルAIデータサイエンス分野に配属された医学科3年生が作成した公開可能な研究成果（Jupyter Notebook，解析スクリプト，ワークフロープログラムなど）が含まれています．これらの資料は教育活動を示すものであり，**学術成果として完成したものではありません**．  


## 概要
本リポジトリは，教育目的の参考資料であり，当研究室における研究実習活動の記録です．  


## 免責事項
全ての資料は教育目的で作成されており，公開が認められていない情報（同意が必要なヒトコホートデータ，未公開の動物実験データ，未発表プロジェクトの詳細）は含まれていません．内容は公開可能な要素に限定され，簡略化されている場合があります．また，学生が実習中に作成した成果物であるため，誤りや意図しない挙動を含む可能性があり，臨床や実運用を目的とした利用はできません．
