# 2025年度医学研究実習

---

このリポジトリは、医学研究実習において実習生が作成した **コード（解析スクリプト・実験プログラムなど）** を管理するためのものです。  
**main ブランチ** にはリポジトリの概要と、各実習生のブランチへのリンクをまとめています。  

## リポジトリ構成
- **main ブランチ**  
  - この README（リポジトリ概要とブランチ一覧）  

- **実習生ごとのブランチ**  
  - 実習に関連するコード

## ブランチ一覧
各実習生のコードは以下の専用ブランチに保存されています。

- [IkumaFukumoto ブランチ](../../tree/IkumaFukumoto)  
  - Ikuma Fukumotoのコード  

- [NatsukiNagae ブランチ](../../tree/NatsukiNagae)  
  - Natsuki Nagaeのコード  

- [HirofumiMitani ブランチ](../../tree/HirofumiMitani)  
  - Hirofumi Mitaniのコード
